/父与子的编程之旅：与小卡特一起学Python/ 不完全笔记

发表于 2018-01-03

花了一学期水了门Python，结果发现讲的内容还没给小孩看的课本多，所以圣诞假的时候瞄了几眼写了点笔记。
教程中使用的是2.7版本

List

向列表添加元素

  • append()
    • extend()
  • insert(index,element)

    从列表删除元素

  • list.remove(element)
  • del list[index]
  • lastLetter = list.pop(index) //将列表中的元素挪走 留空时挪走最后一个

    搜索列表

  • index = list.index(element)

    列表排序

  • list.sort()
  • list.reverse() //逆置
  • list.sort(reverse=True) //逆向排序
  • sortedList = sorted(list)

Dictionary

  • value = dictionary.values()

Function

  • global var //强制全局

Object

  • object.attribute
  • object.method()

Module

  • import my_module
  • from my_module import my_function
  • from my_module import* //从模块导入所有名字
  • time
    • sleep(seconds)

打印格式化与字符串

换行

  • 不换行：语句后加逗号or字符串间用加号连接
  • 换行符：\n

    水平间隔制表符（Tab）

  • 制表符：\t
  • print “s1/ts2” //s2部分将挪到下一个制表点（每八个字符为一个制表快）

    格式化

  • print “123’,str,’456”
  • 浮点数格式化时四舍五入，整数格式化时直接去掉多余位
  • %+f //总是显示正负号
  • %e or %E //数字的e记法
  • %g or %G //自动选择浮点数或E记法
  • 打印百分号： %%
  • format()
    print ‘I got {0:.1f} in math, {1:.1f} in science’.format(math,science)
